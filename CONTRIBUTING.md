# Contributing

`codenzia/laravel-superadmin` is an internal Codenzia package, published
publicly on Packagist so it can be installed in client projects without
authentication setup.

## We do not accept external contributions

This package is developed and maintained exclusively by the Codenzia team.
Pull requests from outside contributors will not be reviewed or merged.

If you found a problem or want a feature:

- **You can fork the repository under the MIT license** and maintain your
  own version. That's the whole point of MIT.
- **Bug reports and feature requests are not accepted** via Issues, PRs,
  or other channels. Issues are disabled on this repository.

If you're a Codenzia team member, see the internal handbook for the
contribution workflow.
